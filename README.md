OpenEmbedded/Yocto layer for Ubuntu Snappy support
==================================================

This meta layer adds support for Ubuntu Snappy for all OpenEmbedded/
Yocto based devices.

This meta layer mainly contains the following components which are
required for Snappy:

 * snapd
 * snap-confine
